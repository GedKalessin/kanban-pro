# 🎯 Kanban Pro for Obsidian

Advanced Kanban board plugin for Obsidian with a modern, sleek UI inspired by ClickUp.

## ✨ Features

### Multiple Views
- **Board View**: Classic Kanban board with drag & drop
- **List View**: Compact list of all tasks
- **Timeline View**: Horizontal timeline with date-based scheduling
- **Gantt View**: Professional Gantt chart for project planning
- **Roadmap View**: Milestone-based planning 

### Advanced Features
- 🏊 **Swim Lanes**: Group cards horizontally by assignee, priority, or custom tags
- 🎯 **WIP Limits**: Set work-in-progress limits per column
- ✅ **Checklists**: Add subtasks to cards
- 🔗 **Note Linking**: Link cards to Obsidian notes
- 🎨 **Custom Colors**: Colorize columns, cards, and swim lanes
- 🔄 **Undo/Redo**: Full history support
- 🔍 **Advanced Filters**: Filter by assignee, priority, tags, dates
- 📊 **Analytics**: View board statistics and insights

### Modern UI/UX
- Clean, modern design inspired by ClickUp
- Smooth animations and transitions
- Responsive layout
- Dark/light theme support
- Keyboard shortcuts
- Quick actions and context menus

## 🚀 Getting Started

### Installation

1. Open Obsidian Settings
2. Go to Community Plugins and disable Safe Mode
3. Click Browse and search for "Kanban Pro"
4. Install and enable the plugin

### Creating Your First Board

1. Click the Kanban icon in the ribbon or use command palette
2. Choose "Create new Kanban board"
3. Select a template or start with a blank board
4. Start adding cards!

## 📋 Templates

Choose from pre-built templates:
- **Basic Kanban**: Simple 3-column board
- **Agile Sprint**: Scrum-style sprint board
- **Software Development**: Complete dev workflow
- **Personal Tasks**: Daily task management
- **Project Management**: Multi-phase projects
- **Content Pipeline**: Content creation workflow

## ⌨️ Keyboard Shortcuts

- `Cmd/Ctrl + N` - Create new card
- `Cmd/Ctrl + K` - Quick search
- `Cmd/Ctrl + F` - Filter cards
- `Cmd/Ctrl + Z` - Undo
- `Cmd/Ctrl + Shift + Z` - Redo
- `Cmd/Ctrl + S` - Save board
- `Escape` - Close modal/deselect

## 📖 Usage Tips

### Organizing with Swim Lanes
Group cards horizontally by:
- Assignee (team members)
- Priority (critical, high, medium, low)
- Custom tags (features, bugs, etc.)

### Using Filters
Quickly find cards by:
- Search query
- Assignee
- Priority level
- Tags
- Due date range
- Completion status

### Time Tracking
- Start timer on any card
- View time spent in analytics
- Export time reports

## 🔧 Development

### Building from Source

```bash
# Clone the repository
git clone https://github.com/yourusername/obsidian-kanban-pro

# Install dependencies
npm install

# Run development build with hot reload
npm run dev

# Build for production
npm run build
```
