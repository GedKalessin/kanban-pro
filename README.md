## ğŸš€ Getting Started

### Installation

1. Open Obsidian Settings
2. Go to Community Plugins and disable Safe Mode
3. Click Browse and search for "Kanban Pro"
4. Install and enable the plugin

### Creating Your First Board

1. Click the Kanban icon in the ribbon or use command palette
2. Choose "Create new Kanban board"
3. Select a template or start with a blank board
4. Start adding cards!
